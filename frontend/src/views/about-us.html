What is Transcendance ?

Transcendence is a web application project developed as part of the 42 School curriculum. The project focuses on creating a single-page application around the game of Pong with multiplayer capabilities.

Main Features

  • • Single-page application
  • • Real-time multiplayer Pong game
  • • User authentication via Google Sign-In
  • • User profiles
  • • Match history
  • • Simple Player statistics

The project also implements mandatory security features and uses modern web technologies to create a functional gaming platform.
